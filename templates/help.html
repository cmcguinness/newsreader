Newsreader Help

Mostly, this app is intended as a demo, so the UI leaves a lot to be desired. The basics of it are as follows:

  • When you start the app, it retrieves headlines from CNN Lite.
  • Any that it hasn't seen before get assigned tags to help with scoring
  • The highest scored stories you haven't read are shown first.
  • If you like or dislike a story, that is used in future scoring.
  • If you click on a headline, you are presumed to have "liked" the story and the story opens in a new tab.

Apart from that, you can learn more about the app by visiting McGuinness on AI or by reading the README.md in the repository.
